# Prediksi Konsumsi Listrik Rumah Tangga

## Deskripsi

Project ini merupakan implementasi Machine Learning menggunakan Python untuk memprediksi konsumsi listrik rumah tangga.

## Dataset

Dataset yang digunakan:

Household Power Consumption Dataset

Sumber:
Kaggle - Electric Power Consumption Data Set


## Metode Machine Learning

Metode yang digunakan:

1. Linear Regression
2. Random Forest Regression


## Library Python

Library yang digunakan:

- Pandas
- NumPy
- Scikit-Learn


## Fitur Dataset

Variabel yang digunakan:

- Global_active_power
- Global_reactive_power
- Voltage
- Global_intensity
- Sub_metering_1
- Sub_metering_2
- Sub_metering_3


## Tujuan

Membuat model yang dapat memperkirakan konsumsi listrik berdasarkan data penggunaan listrik sebelumnya.
